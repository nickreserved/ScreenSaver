Screen Saver του Παύλου Γκέσου
――――――――――――――――――――――――――――――
Προγραμματίστηκε 100% σε Assembly!!!

Φτιαγμένο με τον NASM Assembler και ALINK Linker.

Γράφτηκε τον Απρίλιο του 2002, σε μια προσπάθεια να φύγω από την Assembly για DOS και να πάω σε Assembly για Windows.


1. Μετονομάστε το αρχείο ScreenSaver.exe σε ScreenSaver.scr
2. Αντιγράψτε το στον φάκελο c:\windows\system (Win9x/WinME) ή c:\winnt\system32 (WinNT/Win2000)
3. Από Πίνακα Ελέγχου -> Οθόνη -> Προστασία οθόνης επιλέξτε τον
4. Ρυθμίστε τον
5. Καλή απόλαυση!!!


Αν θέλετε να το κρατήσετε EXE να ξερετε ότι:
ScreenSaver.exe /C		Ρύθμισε τον Screen Saver
ScreenSaver.exe /A		Ρύθμισε το Password του Screen Saver


Η Ρύθμιση του Screen Saver δημιουργεί στη Registry το κλειδί:
HKEY_LOCAL_MACHINE\Software\Gessos Screen Saver\
όπου υπάρχουν αποθηκευμένες οι διαστάσεις του τετραγώνου που μετακινείται.
